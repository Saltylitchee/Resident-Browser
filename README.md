Doppel

「誰の、どんな不便を、どう変えるか」
「作業を中断せずに情報をストック・閲覧したい」


How to use:
・任意のWebサイトからURLを取得する(Alt + C)
・取得したURLからデスクトップ移動についてくる小窓を生成する(Alt + V)
・


ShortCuts:
1, Alt + C: URLを記憶（通知が出る）
2, Alt + V: 取得したURL小窓で生成
3, Alt + R: ページの再読み込み
4, Alt + W: インジケーター化
5, Alt + S: 小窓状態とインジケーター状態の切り替え
6，Alt + D: 小窓の位置とサイズの切り替え 
7，Ctrl + F: 検索バーの表示(Google検索 or ページ内検索)
